Project Details


Using ScraperAPI and BeautifulSoup, eBayScraper periodically scrapes product data, such as price, date, and condition, from eBay.com for a variety of items.
The program utilizes asynchronous programming to organize concurrent API requests, speeding up the scraping process.
Use pandas within a Jupyter environment to compare the performance between groups of items. eBayScraper identifies item groups which depreciate the quickest, sell in large quantities, fluctuate seasonally, and have other informative characteristics.
