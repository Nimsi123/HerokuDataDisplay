Project Details


This website marks the end of a 3-stage data pipeline, scraping, data analytics and graphing, and visualization. The pipeline is scheduled to run once a month, repeatedly updating a database with product data such as price, date, and sale conditions.
Uses Deta Base and Deta Drive to store product data and data visualizations.
